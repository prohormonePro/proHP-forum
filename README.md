## Architecture
- **Frontend:** React (Vite), Zustand, Tailwind CSS
- **Backend:** Node.js, Express, PostgreSQL
- **Infra:** Nginx, systemd, Ubuntu

## The 3-State Monetization Engine
All gating is enforced **server-side** (frontend is cosmetic).

1. **Window Shopper (Unauthenticated):** API strips critical data. UI degrades into an email capture gate.
2. **Lead (Cookie Auth):** Email captured. API serves partial intel. UI injects live Stripe checkout CTAs.
3. **Inner Circle (JWT Auth):** Payment verified; full compound intel and community access unlocked.
